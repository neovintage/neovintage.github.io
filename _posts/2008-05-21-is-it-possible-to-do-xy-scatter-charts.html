You Can! I've forked Gruff on github and written some code to give gruff the ability to plot X and Y points on a graph. Although, I can't take all of the credit for this. Topfunky started the ball rolling and I just took it a little further. I'm hoping to get this committed to the main branch soon.

Get the Code:

As of this point the code is only in my forked github repository. You'll have to download the code from github and create a gem if you'd like to use it.

git clone git://github.com/neovintage/gruff.git
rake gem


Example:

Let's run through an example of how to create the above chart. The code will look very similar to the sample provided on nubyonrails.com
scatter_chart = Gruff::Scatter.new(400)
g.title = "Basic Small Scatter Chart Test"

# Parameter 1 is the name of the dataset
# Parameter 2 is the values for the X-Axis
# Parameter 3 is the values for the Y-Axis
g.data('Chuck', [20,10,5], [5,10,20])
g.data('Brown', [10,12,14,8], [12,6,10,8] )
g.data('Lucy', [8,16,8], [8,10,6] )

g.write('basic_scatter.png')

The only thing that you can't do with the Scatter graph is set labels for the X-Axis. Maybe that could be a future enhancement.
